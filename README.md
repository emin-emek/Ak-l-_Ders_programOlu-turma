# Ak-l-_Ders_programOlu-turma 
Proje Amacı:
Üniversitelerin ders programlarını otomatik olarak oluşturan ve yönetebilen web tabanlı bir akıllı ders planlama sistemi.Tüm bölümlerin öğretmenlerini,sınıf,
dersi,aykırı günlerini,aykırı saatlerini yazıpı tek tuşla tüm bölümümlerin ders programlarını veririr.

Ana Görevler:
Ders Programı Otomatik Oluşturma - Öğretmen, sınıf, zaman dilimi çakışmalarını göz önüne alarak optimal program oluşturma
Bölüm Yönetimi - Üniversite bölümlerini, öğretmenleri, sınıfları ve dersleri yönetme
Çatışma Çözümleme - Öğretmen uygunluk saatleri, sınıf kapasitesi vs. kontrolü
Gerçek Zamanlı Senkronizasyon - WebSocket üzerinden canlı veri güncellemeleri.
